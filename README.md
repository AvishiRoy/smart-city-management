# Smart City Management System

## Overview

An AI-powered Smart City Traffic Management System that predicts traffic congestion using Machine Learning and provides insights through a web dashboard.

## Features

* Traffic Volume Prediction
* Flask REST API
* Flask backend API
* Interactive dashboard
* Data visualization
* Machine Learning integration
